## コマンドメモ

```sh
# 投稿の追加
npm run ng generate @scullyio/init:post --name="{title}"

# ルーティング検出
npm run scully

# ローカルで確認
npm run scully:serve

# SCSSとかアセット更新した時
npm run build
```
