object型は追加のプロパティを許容する。

```ts
const v1: object = { a: 1 };
```

Record型は追加のプロパティを許容しない。

```ts
type T1 = Record<string, never>;

const v1: T1 = {};
const v2: T1 = { a: 1 }; // Error
```
