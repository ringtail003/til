## Unionの一部を抽出する

```ts
type T1 = 'a' | 'b' | 'c';

type T2 = Extract<T1, 'a' | 'b'>;
// 'a' | 'b'

type T3 = Extract<T1, 'a' | 'zzzzz'>;
// 'a'
// 誤ったテキスト指定は利用側で使えず実装時に間違いに気づける
```

## オブジェクトのUnionを特定する

```ts
type T4 = { type: 'a'; b: string; } | { type: 'b'; b: number };

type T5 = Extract<T3, { type: 'a' }>;
// { type: 'a'; b: string; }
```

## プロパティにUnionに定義されたテキストを使うことを強制する

```ts
type T1 = 'a' | 'b';

type T2 = {
  type: Extract<T1, 'a'>;
};

const v1: T2 = {
  type: 'a'
};
const v2: T2 = {
  type: 'a'
};
```
