複数の要素指定をまとめて書くことができる。

```css
  /* 
  h1 { ... }
  section { ... }
  */

  :is(h1, section) {
    background: lightgray;
  }
```

```css
  /* 
  h1 span:hover { ... }
  section span:hover { ... }
  */
  :is(h1, section) span:hover {
    background: yellow;
  }
```

疑似要素（`:first-child` など）には無効。
