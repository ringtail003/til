```shell
# 1.2.3 => 1.3.0-alpha.0
npm version preminor --preid alpha

# 1.3.0-alpha.0 => 1.3.0-alpha.1
npm version prerelease

# 1.3.0-alpha.1 => 1.3.0-rc.1
npm version prerelease --preid rc

# 1.3.0-rc.1 => 1.3.0
npm version patch
```

バージョンにvが付くように設定する

```shell
// .npmrc
tag-version-prefix="v"
```
